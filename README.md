# 🤖 Chatboot com IA

Chatbot desenvolvido em Python, com interface web criada usando **Streamlit** e respostas geradas pela **API da OpenAI (GPT-4o)**.

## 📋 Sobre o projeto

Aplicação de chat simples onde o usuário digita uma mensagem, ela é enviada para o modelo de IA da OpenAI, e a resposta é exibida na tela em formato de conversa, mantendo o histórico da sessão.

## 🚀 Funcionalidades

- Interface de chat interativa (estilo WhatsApp/ChatGPT)
- Histórico de conversa mantido durante a sessão
- Integração com a API da OpenAI (modelo GPT-4o)
- Indicador de carregamento enquanto a IA responde
- Botão para limpar o histórico de conversa
- Avatares personalizados para usuário e IA

## 🛠️ Tecnologias utilizadas

- [Python](https://www.python.org/)
- [Streamlit](https://streamlit.io/) — frontend e backend em um só framework
- [OpenAI API](https://platform.openai.com/) — geração das respostas
- [python-dotenv](https://pypi.org/project/python-dotenv/) — gerenciamento seguro de variáveis de ambiente
